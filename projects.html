Conducted a full black-box penetration test on a live academic portal. The engagement covered the complete attack lifecycle — passive & active reconnaissance, vulnerability identification, exploitation, and stakeholder reporting. The most critical finding: privilege escalation from a student account to professor-level access via broken access control, which would have exposed the entire student record database in production.

Built a fully functional simulated banking application with intentional security vulnerabilities for use in penetration testing labs. The app replicates real banking features — user/admin login, account management, fund transfers — while deliberately exposing SQLi, XSS, and IDOR flaws. Used as a hands-on teaching tool to give students a safe environment to find, exploit, and document real vulnerabilities before working on live engagements.
