Fake Bank Portal

The Fake Bank Portal is a simulated online banking application developed using Flask and SQLite. It replicates essential banking features such as user and admin login, account management, and password updates.

This project is intentionally designed with SQL injection vulnerabilities, making it a safe environment for practicing penetration testing and learning how real-world attacks work.

In addition, the portal also includes XSS (Cross-Site Scripting) scenarios to demonstrate how malicious scripts can be injected and executed within a web application. This allows hands-on experience in identifying and mitigating client-side security flaws.

By working on this project, I demonstrated my skills in web application development, database integration, and secure coding practices, while also applying offensive security techniques to showcase practical cybersecurity knowledge.
